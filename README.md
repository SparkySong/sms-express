# 简讯速递微信小程序

这是一个微信小程序新闻阅读平台，为用户提供新闻浏览、分类查看、收藏等功能。

## 项目概述

简讯速递是一款专注于提供快速、简洁新闻阅读体验的微信小程序。用户可以浏览各类新闻，根据分类查看不同领域的新闻，还可以收藏感兴趣的文章，方便后续阅读。

## 功能特性

- **新闻浏览**：首页展示推荐新闻、头条新闻和分类新闻
- **分类查看**：支持按不同分类筛选新闻
- **文章详情**：查看新闻详细内容
- **用户系统**：支持注册、登录功能
- **个人中心**：用户信息管理
- **收藏功能**：收藏感兴趣的文章
- **历史记录**：记录阅读历史
- **搜索功能**：搜索感兴趣的新闻
- **消息通知**：接收系统通知
- **文章发布**：用户可以发布新闻文章

## 技术栈

- 前端：微信小程序原生框架
- 后端：Node.js + Express
- 数据库：MySQL
- 认证：JWT

## 开发与部署

### 开发环境要求

- 微信开发者工具
- Node.js (v14+)
- MySQL (v5.7+)

### 运行步骤

1. 克隆项目到本地
2. 安装依赖：`npm install`
3. 按照 server/README.md 中的指引配置后端服务
4. 在微信开发者工具中导入项目
5. 修改 app.js 中的 BASE_URL 为你的服务器地址
